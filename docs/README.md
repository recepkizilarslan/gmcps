# Documentation

This folder contains the project documentation for **GMCPS** (OpenVAS/GVM MCP Server).

- **Installation**: `docs/INSTALLATION.md`
- **Development**: `docs/DEVELOPMENT.md`
- **Tool contracts (inputs/outputs)**: `docs/TOOLS.md`
- **Use-case playbook (prompts + tool flows)**: `docs/USE_CASES.md`

If you are new to the project:

1. Follow `docs/INSTALLATION.md` to run the server and verify the SSE endpoint.
2. Open `docs/TOOLS.md` to see the exact tool schemas and common call format (`arguments.input`).
3. Use `docs/USE_CASES.md` as a prompt/workflow reference for real-world scenarios.
